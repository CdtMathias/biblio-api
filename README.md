Bonjour, je vous remercie pour votre confiance.

Tout d'abord voici le lien de l'API : https://web-production-c4b0f.up.railway.app/docs#/default/return_loan_by_id_loans_return__id__put

Pour l'utiliser vous devez tout d'abord créer un compte à partir de POST Register en renseignant username et password.
Vous pouvez ensuite utiliser POST Login pour vous connecter, une fois cela fait vous pouvez cliauer en haut de la page sur "Authorize" pour vous connecter à nouveau avec les mêmes identifiants.

Tout se passe dans les commandes visible:

GET pour observer et obtenir les données, POST pour publier ou ajouter, PUT pour modifier et DELETE pour supprimer.

Members representent vos adherants, Books vos livres et Loans les locations.
